13Dプリンター導入の検討ポイント

講師自身が4店舗目（新事業）として3Dプリンターの購入を検討中。機種選びで悩んでいるリアルな思考プロセスが共有されました。

  • 3Dプリンターには大きく2種類ある：よく見るプラスチック（フィラメント）タイプと、ツヤツヤで可愛いレジンタイプ。
  • レジンタイプは見た目が可愛いが、ほぼ海外製。日本語の説明書やサポートがなく、「数週間で壊れた」というレビューもあり、約30万円の投資としてはリスクが高い。
  • 結論の方向性：みんなが使っている定番機種を選ぶ。情報が多く、困ったときにすぐ調べられる・聞ける。
ポイント：CapCutやApple製品と同じ理屈。ユーザーが多いツールは情報量が多く、トラブル時の解決が早い。珍しい機材をあえて選ぶメリットは少ない。

2売れる商品の考え方

「自分が絶対欲しい」は最強のリサーチ

講師が3Dプリンターで作りたいのは「ズボンの裾を留めるアイテム」。自分が毎日困っていて、すでに類似品をお金を出して買っている＝同じ悩みを持つ人が確実に存在するという確信につながっています。

「私がこんなにも欲しいと思うからには、みんなも欲しいと思うはず」

「可愛い」だけでは買われない

  • 悩み解決 × 可愛いが最強の組み合わせ。可愛いだけだと「可愛いけど要らない」「可愛いけど高い」で終わってしまう。
  • 「こういう時も使える」「これにも使える」という購入の言い訳（理由）が1つあるだけで、購入ハードルは大きく下がる。

リサーチの目線

  • PinterestやInstagramで「バッグ」「キーホルダー」などカテゴリ名で検索するのではなく、好きなジャンルのインテリアやファッションの写真から「暮らしの中でこれがあったらいいのに」を探す。
  • 「作りたいものを作る」ではなく「今売れている商品を徹底リサーチして、それを3Dプリンターで作れないか」と考える視点も有効（例：スマホスタンド、スマホカバー）。
お金に働いてもらう視点：3Dプリンターは最初の設計だけ手間がかかるが、あとは機械が作ってくれる。「自分の手を使わずに商品ができる仕組み」として捉えると投資価値が見えてくる。

32店舗目・新商品展開のヒント

受講生の相談：メンズ向けポケットチーフ

  • ポケットチーフを持つ層は上品でお金に余裕がある可能性が高く、単価を上げやすい。
  • ただし「お金持ちそうな男性が欲しがるアカウント」の世界観づくりが必要になる。
  • 発展案：ポケットチーフ単体ではなく「女性から男性へのギフトショップ」として展開。ネクタイ・ハンカチとのセット販売も好相性。
  • 既存のスマホショルダー店は女性客が多いので、父の日・バレンタインなどのタイミングでテスト販売すれば、「自分の分と一緒に買おう」という流れが作れる。
  • まずは出してみて反応を見てから決める。作るのも早い（約10分）ならなおさら試す価値あり。

複数店舗を持つメリット

  • 1店舗だけだと売上に一喜一憂・執着しがち。複数あればどれかの売上が悪くても悩んでいる暇がなく、行動し続けられる。
  • 学ぶ前に作った1店舗目より、売り方を学んだ今作る2店舗目のほうが売りやすい商品設計にできる可能性がある。

売れる商品か、作りたい商品か

すでに売上が立っている（30〜50万円台の）人なら、ときめきを感じる商品に挑戦するのも十分あり。「楽しそう」という気持ちは継続の大きな原動力になる。

4これからのハンドメイド業界と「売る力」

  • 3Dプリンター、アクリルカッター、刺繍機など技術がなくても作れる機械がどんどん増えている。
  • 暮らし系・インテリア系インフルエンサーなど、「Instagramを伸ばすのがうまい人たち」がハンドメイド的なジャンルに参入してきている（例：Threadsで話題のビーズ販売者も元・暮らし系発信者）。
  • 純粋なハンドメイド作家として始めた人は、今後Instagramを伸ばすのが難しくなっていく可能性がある。
「どれだけ商品が良くても、どれだけ技術があっても、売るのがうまい人に負けてしまう。だから――売るのがうまくなりましょう」

5再販の戦略

  • 秒で売り切れることが確定している回：最初から出せる限界の数を「30個限定」のようにドンと出して売り切るほうがよい。
  • 売り切れるか怪しい回：数を少なめに出して、再販で動きを作るほうが効果的。
  • 毎回再販していると、ずっと見ているフォロワーは「どうせまた再販するでしょ」という心理になり、購入の緊急性が薄れる。
  • 「もう買えないかもしれない」というお店のほうが買いたくなる。やり方を毎回変えることで飽きずについてきてもらえる。
  • 再販するか迷ったら：欲しい人がたくさんいる＝飽きられていない証拠なので再販してOK。何度も再販が続いているなら、次回から最初の数量を増やして調整する。事前に温度感を測れるようになることが大事。

6キャプションとの付き合い方

  • 販売会前の時期は、認知目的のリールでも販売会告知キャプションのコピペでOK。講師自身も毎回コピペ運用。
  • キャプションは「書いたほうがいいけれど、そこに時間をかけて頑張る場所ではない」。余力があるならストーリーズを優先すべき。
  • キャプション作成はClaudeなどのAIに考えてもらうのも手。

7税理士の選び方とお金の管理

選び方の基準

  • まず自分の目的を明確にする：確定申告だけ頼みたいのか、節税したいのか、利益目標に向けて相談しながら進めたいのか。
  • ハンドメイド・物販・小売業に強い税理士を探す。ただし「〇〇専門」は言ったもの勝ちの面もあるので、実際に話を聞きに行って相性を確かめるのが大事。
  • 対応が雑だったり「なめられている」と感じたら、お金を払ってまで頼む必要はない。合わなければやめてよい。
  • 知り合いの経営者が契約している税理士を紹介してもらうのも有力な方法（講師も紹介経由）。

そもそも税理士は必要か

  • ハンドメイドはお金の流れがシンプル（入り：BASE・対面販売／出：材料費・広告費）。売上100万円程度なら税理士なしでも対応可能という話も。
  • AIを活用すれば確定申告は自力でできる：「完璧に確定申告できるように1ステップずつ案内して」と依頼し続ける。役所の書類も写真を撮って「これは何の書類？何を書けばいい？」と聞く。
  • 重要な判断は複数のAI（Claude・Gemini・ChatGPTなど）に同じ質問をして答えを突き合わせる。異なる回答が出たら「他のAIはこう言っているが、なぜあなたはこう考える？」と議論させると信憑性が上がる。

8迷惑客・クレームへの対応

実例と対処

  • パクリ報告DM（「同じ内容のリールを見ました。残念です。一応報告しました」）→ 即ブロックせず放置も選択肢。相手にする価値のないものは深追いしない。
  • タメ口・馴れ馴れしいDM → ストーリーズへの反応はハートやスタンプのみで返し、会話が始まらないようにして距離を作る。質問には丁寧に返す。「DMは1対1＝丁寧に」を貫く。
  • 開封後の雑な返品 → 対策として「返品は未開封に限る」と明記する。
  • 買う買う詐欺（買うと言い続けて買わない人） → 徐々に壁を作ってよい。

心構え

  • お客さんが増えるほど「やばい人」に当たる確率も上がる。それが普通だと知っておく。
  • 自分が買い物をするときも「向こう側に人がいる」ことを忘れない（講師自身、海外仕入れサイトで返金依頼をしたらブロックされた経験談も）。

9モヤモヤしたときの切り替え方

  • 人に話して笑い話にする：スタッフや友人にマシンガントークで話し、何度も話して「面白い話」に精度を高めていく。
  • 笑い話にできない小さいモヤモヤは意識的に忘れる：「他人から聞いたらしょうもない話」レベルのものは「忘れよう」と決めてなかったことにする。
  • 悪口やアンチはスクショだけ取っておく：悔しいまま終わらせず、いつか笑い話や仕事のネタ（種）にする。
  • AIに話すときはひと工夫：AIは肯定ばかり／解決したがりになりがち。「寄り添わないで」「公平に見て」「褒めないで」と前置きすると、ちょうどいい壁打ち相手になる。

今日の重要ポイントまとめ

  • 機材・ツールは「みんなが使っている定番」を選ぶ。情報量がサポートになる。
  • 売れる商品＝「自分が心底欲しい悩み解決アイテム」×「購入の言い訳が作れる商品」。
  • 2店舗目は、既存客層からの導線（ギフト需要・イベント）を活かして小さくテスト。
  • 商品力より売る力。これからの時代は「売るのがうまくなる」ことが最大の防御。
  • 再販は毎回しない。売り切れ確実なら限定数をドンと、迷うなら少なめ＋再販で。
  • キャプションよりストーリーズ。手を抜く場所を決めて、力を入れる場所に集中する。
  • お金まわりは目的を明確に。小規模ならAI活用で自力申告も可能。複数AIで検証を。
  • 迷惑客には距離を作る仕組み（未開封限定・スタンプ返し）で消耗を防ぐ。
  • モヤモヤは「笑い話」か「ネタ」に変換。できないものは忘れると決める。
